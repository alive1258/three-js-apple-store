## Getting Started Installation

First, run the development server:

```bash
# Clone the repo
git clone https://github.com/your-username/apple-store-3d.git

# Go to folder
cd apple-store-3d

# Install dependencies
npm install

# Start development server
npm run dev

```

## Build for Production

```bash
npm run build
npm run preview


```
